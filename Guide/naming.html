CSS 파일 네이밍

CSS 성격에 맞는 파일명을 작성한다.

파일명만으로 어디서 사용되는 CSS인지 파악 될 수 있게 작성한다.

파일명은 한단어로 작성하는 것을 지향한다.

web.css / user.css / mobile.css / agent.css / admin.css / style.css 등

파일명으로 어떤 플러그인의 CSS인지 파악이 되지 않는 경우 CSS 파일명을 수정한다.

all.min.css > 어떤 파일인지 파일명으로 파악할 수 없기 때문에 수정 가능 수정 후 : fontAwesome.css

플러그인 CSS 파일명 수정 방법

플러그인명은 낙타표기법을 사용한다. 하이픈(-)으로 플러그인의 유형을 표기한다. 없는 경우 생략한다. 온점(.)으로 플러그인의 버전을 표기한다. 버전 앞에는 하이픈(-)을 넣는다. 없는 경우 생략한다. CSS 축약 파일인 경우에는 .min을 표기한다. ex)fontAwesome-style-(생략가능)-.1.2.1.min.css
