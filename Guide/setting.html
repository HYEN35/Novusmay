페이지 작업 순서

페이지 작업은 웹 흐름, 메뉴 순으로 작업한다.

ex) 로그인 > 메인 > 메뉴1 > 메뉴2 ...

index.html/inventory.html(이하 index.html) 작성법

프로젝트 작업 전 index.html을 작성한다.

index.html에는 프로젝트명, 작업자, 작업일, 홈페이지 유형을 작성한다.

작업된 모든 html 파일을 리스트업 한다.

index.html 파일명은 서비스(또는 솔루션)과 홈페이지의 파일명과 다르게 만든다.

서비스의 경우 : index.html

홈페이지의 경우 : inventory.html

홈페이지에서 index.html은 메인페이지 파일명으로 작업한다.

W3C 검사하기

W3C 검사는 필수로 진행한다.

W3C 검사는 한 페이지의 작업이 완료 된 후 한다.
