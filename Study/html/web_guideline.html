웹 접근성 지침 소개


WCAG W3C에서 발표한 웹 콘텐츠 접근성 지침.
KWCAG 해외 웹 표준 기술 동향을 토대로 국내 설정에 맞게 반영된 한국형 웹 콘텐츠 접근성 지침. 원칙 4개, 지침 13개, 항목 24개로 구성.

1. 적절한 대체 텍스트 제공

텍스트가 아닌 콘텐츠는 그 의미나 용도를 인식할 수 있도록 대체 텍스트를 제공해야합니다.
텍스트가 있는 이미지를 이미지 요소로 마크업할 때 대체 텍스트는 이미지 요소의 alternative속성인 'alt'에 적용해주시면 됩니다.

2. 자막 제공

멀티미디어 콘텐츠에 포함된 음성 및 대사와 동등하게 대체 수단을 제공해야합니다. 대체 수단은 자막, 대본, 원고, 수화 중 하나로 제공하면 됩니다.

- 제 3자가 영상을 업로드 하는 경우, 사용자가 대체 콘텐츠를 제공할 수 있도록 툴을 제공해야합니다.
- 소리는 나오지 않고 텍스트만 나오는 영상일 경우, 따로 원고를 제공하거나 하는 등의 방법으로 대체 콘텐츠를 제공해야합니다.

3. 색에 무관한 콘텐츠 인식

색을 구분하지 못하더라도 콘텐츠를 인식하는데 문제가 없어야 합니다.

- 차트 : 색으로만 구분하는 것이 아니라 패턴이나 굵기 또는 모양이나 테두리 등으로 구분 해주어야 합니다.
- 슬라이드 버튼 : 선택된 버튼을 모양으로 구분하면 명확하게 인식할 수 있습니다.
- pagination : 테두리나 굵기 또는 다른 방법으로 구분이 가능하도록 구현해야 합니다.

4. 명확한 지시 사항 제공

오류사례
- 모양으로만 정보 제공
- 위치나 방향으로만 정보 제공
- 크기로만 정보 제공
- 색으로만 정보 제공
