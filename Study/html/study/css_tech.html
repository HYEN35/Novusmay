1. display:table, display:table-cell, vertical-align:middle을 사용하여 컨텐츠 수직 정렬 하기
- 수직 정렬하려는 컨텐츠에 display:table-cell과 vertical-align:middle 속성을 추가 한 뒤 부모 컨텐츠에 display:table 속성을 추가하면 수직 정렬이 가능하다.
2. float된 컨텐츠 가운데 정렬하기
- float된 컨텐츠를 가운데 정렬 할 때는 display:inline-block, text-align:center 속성을 사용한다. - float된 컨텐츠의 부모에게 display:inline-block 속성을 추가 한 뒤, inline-block 된 부모에게 text-align:center 속성을 추가한다. - inline-block 속성을 사용하게 되면 4px정도의 여백이 생긴다. 이 때 text-align:center 속성이 추가된 컨텐츠에 높이값을 주거나 font-size:0을 선언하면 여백이 사라진다. - height의 값이 고정값이면 height 값을 사용하고, height 값이 유동적으로 변경되어야 하는 경우에는 font-size:0을 사용한다.

3. vw, vh, calc
vw = viewport width vh = viewport height 1vw = 뷰포트 너비의 1% 1vh = 뷰포트 높이의 1% - vw, vh는 실행중인 브라우저의 크기에 맞춰 그 크기를 유동적으로 맞추겠다는 의미이다. - 다양한 브라우저 크기에 대응할 수 있기때문에 반응형 사이트에서 유용하게 사용된다.
calc = calculate - calc은 속성값을 사칙연산으로 계산 할 수 있게 하는 속성이다. - 컨텐츠의 높이를 브라우저에 따라 유동적으로 맞추기때문에 calc 속성과 같이 쓰인다.

4. 아이폰 노치 대응하기
아이폰 노치란? - 아이폰 X부터 추가된 영역으로, Safe-area라고 불린며, 상/하단에 여백이 생기는 것을 말한다. - 아이폰 노치는 별도의 속성값으로 대응해줘야 한다. - 별도의 속성값을 추가하지 않으면 노치영역에서 레이아웃이 겹치거나 잘리는 현상이 발생한다. - 보통 body에 env(), constant()를 추가하고 하단 네비버튼에 추가한다.
속성 적용하기 1. 뷰포트 메타태그에 viewport-fit:cover를 추가한다. 2. body에 env(), constant() 속성 추가한다.
