3. Mac 스크린리더 사용법
MacOS voiceOver 스크린리더로 웹접근성 테스트하기
* 맥 운영 체제 모하비 기준
* 맥 os에 기본으로 탑재 되어있어 따로 설치하지않아도 됨
- voiceOver on/off : cmd+f5
- voiceOver 키 : capslock
- 음성 속도 조절 : vo key+shift+cmd+왼쪽 혹은 오른쪽 화살표로 '속도'로 이동한 후 key+shift+cmd+위 또는 아래 화살표로 값 조절
- voiceOver 세부 환경설정(보이스오버 유틸리티) 테스트를 위한 기본 환경 설정
환경설정 → 접근성 → 시각 → 보이스오버 유틸리티 실행 or vo+f8 눌러 실행
