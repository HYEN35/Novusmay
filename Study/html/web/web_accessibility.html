웹 접근성의 이해
2021.01.25 김혜인
1. 웹 접근성이란?
Q. 접근성이란? └ 사용자가 서비스 접근에 편리하게 이용할 수 있는 정도 Q. 웹 접근성이란? └ 웹에 잘 접근할수 있는지 없는지의 정도 └ 어떠한 사용자가 어떠한 기술 환경에서도 전문적인 능력없이 웹사이트에서 제공하는 모든 정보에 접근할 수 있도록 보장하는 것 모든 사용자가 모든 기기에서 웹에 접근할 수 있도록 하는 것! Q. 왜 웹접근성을 지켜야 하는가? 1. 장애인 차별 금지법(장차법) 2. 장애에 구애 없이 모든 사람이 접근할 수 있도록 하는 것이 웹의 필수 요소
2. 장애 환경
* 장애 환경의 종류 - 전맹 시각 장애 - 저시력 시각 장애 - 중증 운동 장애 - 손 운동 장애 - 청각 장애 - 맥에서 윈도우에서만 접근할 수 있게 구현되는 것 - 느린 인터넷 * 장애환경에서 웹을 이용하는 방법 - 스크린리더기 - 점자 정보 단말기 - 화면 확대 기능 사용 - 고 대비 - 보조기기( 헤드포인터, 빅키 키보드, 키카드, 한손사용자용키보드, 트랙볼 마우스 등)을 이용한 키보드 조작 - 안내 문구, 자막
